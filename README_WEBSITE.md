# Veriva Insights Website Foundation

This workspace contains the foundation for a premium UK AI & Data Analytics website:

- `index.html` — page shell structure, sticky header, footer, mobile menu, bottom nav
- `styles.css` — design system, responsive grid, typography, color palette, mobile reset
- `app.js` — JS routing, mobile menu behavior, header scroll behavior, dropdown toggle

The site is intentionally built as a shell-only foundation with page routing and structural components ready for content.
